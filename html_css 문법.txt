이름 : <이름 /> 선택
#id : <이름 id="id" /> 인것을 선택
ex).fancy : fancy라는 id를 가진 것들
A B : A의 하위 원소 중 B를 선택
.classname : <이름 class="classname" /> 인것들을 선택
이름1.classname : <이름1 class="classname" /> 인것들을 선택, 위 문법과의 차이는 특정 이름을 지칭하냐 안하냐 의 차이
A,B : A,B 선택, +라고 생각하면 됨/모든 객체에 대해 선택을 해야되면 *를 통해 쉽게 선택할 수 있음
A + B : A 바로 밑에(A의 그룹을 벗어난뒤) 있는 원소중 B를 선택
A ~ B : A 뒤에 있는 원소(그룹을 벗어난 뒤부터)들 중 B 에 맞는 것들을 선택
A>B : A 바로 아래에 있는(A 그룹안에 있는) B를 선택
